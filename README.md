# 🎮 TicTacToe Web

Modern Tic Tac Toe AI web game built with React, Vite, Tailwind CSS, and Framer Motion.

---

## 🚀 Live Demo

🔗 Play Now:
https://tic-tac-toe-web-7t8d-25my2pbut-mohsen-mohamed-20s-projects.vercel.app

---

## ✨ Features

* Human vs AI
* Human vs Human
* AI vs AI Demo
* Minimax AI Algorithm
* Multiple Difficulty Levels
* Arabic & English Support
* Custom Player Names
* Neon Dark UI
* Smooth Animations
* Responsive Mobile Design
* Scoreboard System
* Game Reset
* Modern Web Experience

## ⚡ Installation

```bash
git clone https://github.com/Mohsen-Mohamed-20/TicTacToe-Web.git

cd TicTacToe-Web

npm install

npm run dev
```

---

## 📦 Build

```bash
npm run build
```

---

## 🚀 Deployment

This project is deployed on Vercel.

Build command:

```bash
npm run build
```

Output folder:

```bash
dist
```
